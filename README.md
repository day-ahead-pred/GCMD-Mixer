# GCMD-Mixer

## Run experiments

The experiment scripts are in the `config` folder.

* `run.sh` is used to run experiments on Linux. An example of usage is

	```bash
	conda activate gcmd
	bash run.sh
	```
* `run.bat` is used to run experiments on Windows. An example of usage is

	```bash
	conda activate gcmd
	run.bat
	```
